Возможности CSS

  1. Широкий спектр параметров форматирования объектов HTML. Средства CSS позволяют:
    • задавать межсимвольный интервал
    • изменять междустрочный интервал
    • форматировать текст: подчеркивать, надчеркивать, перечеркивать;
    • выполнять заливку текста
    • управлять отступом
    • Каждое Слово Начинать С Большой Буквы
    • Управлять размером элементов
  2. Сокращение размера исходного HTML-документа
  3. Настройка объектов HTML: таблиц, списков, гиперссылок и т.д.
